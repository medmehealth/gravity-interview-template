# Gravity Interview Template

## Description

NPM monorepo with a NestJS backend and a sample React frontend.

## Project setup

```bash
npm install
```

## Compile and run the project

```bash
npm start
```

## Run tests

```bash
npm run test
```

## Project Structure

```text
├── backend/        # NestJS backend application
├── frontend/       # React frontend application
```
